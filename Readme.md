# UAS Kecerdasan Buatan

## Judul Proyek
Prediksi Penyakit Tanaman Menggunakan Algoritma Decision Tree dan KNN

## Anggota Kelompok
1. Nama Mahasiswa 1 - NIM
2. Nama Mahasiswa 2 - NIM

## Deskripsi Proyek
Proyek ini bertujuan untuk membangun model kecerdasan buatan yang dapat membantu melakukan klasifikasi penyakit tanaman berdasarkan data gejala.
